# Cloud Security Roadmap

Questo repository documenta il mio percorso per diventare assumibile nel Cloud Computing e nella Cloud Security.

## Obiettivo

Diventare un profilo junior assumibile in Cloud / Cloud Security, con competenze pratiche in:

- Networking
- Linux
- Git e scripting
- AWS
- Infrastructure as Code
- Cloud Security
- Progetti pratici documentati

## Metodo di studio

- 1-2 ore al giorno
- Una fase alla volta
- Pratica prima della teoria
- Tutto documentato su GitHub
- Nessun salto avanti
